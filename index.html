How to Use

  1. Upload an audio file (MP3, WAV, OGG, or M4A format)
  2. Click the play button to start audio playback
  3. Watch the 3D sphere react to your music in real-time
  4. Use your mouse to rotate and zoom the 3D view
  5. Adjust volume and sensitivity for the best experience

Browser Requirements:

This visualizer requires a modern browser with Web Audio API and WebGL support. Works best in Chrome, Firefox, Safari, and Edge.
